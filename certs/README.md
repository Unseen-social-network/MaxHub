# certs/

`russiantrustedca.pem` — сертификат «Минцифры» (Russian Trusted Root/Sub CA),
которым MAX подписывает TLS на `platform-api2.max.ru` (см. рассылку MAX
от июля 2026 про переход с `platform-api.max.ru` на `platform-api2.max.ru`
до 19 июля).

Фактическое доверие этому сертификату для запросов к API MAX уже
обеспечивает библиотека `maxapi` (>=1.2.1) — она сама доверяет
`platform-api2.max.ru` и подмешивает этот же CA в свой SSL-контекст
(`maxapi/client/ssl.py`). Копия здесь — резервная: для ручной установки
в системное хранилище доверенных сертификатов сервера или контейнера,
если это когда-либо понадобится отдельно от библиотеки.

Хост API настраивается явно через `MAX_API_URL` (см. `.env.example`),
по умолчанию — `https://platform-api2.max.ru`.
