# MaxHub Stage 1 — Project Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the MaxHub project skeleton — uv-managed Python 3.12 project, directory layout, pydantic-settings config — so later stages (DB, bot core, features) have a working foundation that installs, lints, and tests green.

**Architecture:** A single `app/` package holding config, handlers, services, db layers per CLAUDE.md's structure. `uv` owns dependency resolution (`pyproject.toml` + `uv.lock`, no requirements.txt). Config is a single `pydantic-settings` `BaseSettings` subclass reading from env/`.env`.

**Tech Stack:** Python 3.12, uv, maxapi[fastapi], fastapi, uvicorn, sqlalchemy[asyncio]>=2.0, asyncpg, alembic, pydantic-settings, apscheduler, aiolimiter, pillow; dev: ruff, pytest, pytest-asyncio.

## Global Constraints

- Python 3.12+, dependency manager is `uv` only — no pip/poetry/requirements.txt anywhere.
- `uv.lock` is committed.
- Messages the bot sends are in Russian; code identifiers/comments in English.
- Env vars required by config: `BOT_TOKEN`, `DOMAIN`, `WEBHOOK_PATH`, `PORT`, `MODE` (webhook|polling), `ADMIN_IDS` (comma-separated), `DATABASE_URL` (`postgresql+asyncpg://...`), `TZ` (default `Europe/Moscow`), `BROADCAST_ACTIVE_DAYS` (default 30), `APP_VERSION`, `GIT_SHA`, `BUILD_TIME`.
- Directory layout must match CLAUDE.md's `## Структура` section exactly (this plan creates the Stage-1-relevant subset; `alembic/`, `deploy/`, `.github/workflows/`, `Dockerfile` are built in later stages).
- Secrets only via env; `.env` is gitignored, `.env.example` is committed with placeholder values.

- [ ] **Step 3: Write the implementation**

```python
# app/config.py
from functools import lru_cache
from typing import Literal

from pydantic import field_validator
from pydantic_settings import BaseSettings, SettingsConfigDict


class Settings(BaseSettings):
    model_config = SettingsConfigDict(
        env_file=".env",
        env_file_encoding="utf-8",
        extra="ignore",
    )

    bot_token: str
    domain: str
    webhook_path: str
    port: int
    mode: Literal["webhook", "polling"]
    admin_ids: list[int]
    database_url: str
    tz: str = "Europe/Moscow"
    broadcast_active_days: int = 30
    app_version: str = "dev"
    git_sha: str = "unknown"
    build_time: str = "unknown"

    @field_validator("admin_ids", mode="before")
    @classmethod
    def _parse_admin_ids(cls, value: object) -> object:
        if isinstance(value, str):
            return [int(part.strip()) for part in value.split(",") if part.strip()]
        return value


@lru_cache
def get_settings() -> Settings:
    return Settings()
```
